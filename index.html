<div>: Uma tag genérica usada para agrupar elementos e estruturar o layout de uma página.

Abas (Tabs): Organizam conteúdo em seções sem recarregar a página.

CSS:

  • background-color: define a cor de fundo
  • font-size: define o tamanho da fonte
  • text-align: controla o alinhamento do texto
  • color: define a cor do texto
  • text-transform: controla letras maiúsculas/minúsculas
